# azk

A very experimental version of `azk` written with [crystal](crystal-lang.org), ignore it and continue forward for [azk](https://github.com/azukiapp/azk).

### A development environment orchestration tool

1. Install [crystal](http://crystal-lang.org/docs/installation/index.html)
2. Install [shard](https://github.com/crystal-lang/shards)
3. Build from the source:

```sh
$ git clone https://github.com/nuxlli/azk-crystal azk
$ cd azk
$ crystal build src/azk.cr -o bin/azk
$ ./bin/azk
```
